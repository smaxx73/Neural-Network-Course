Core Concepts

Bias, activation functions, vectorized computation — Questions 1 to 4

Q 01 / 15 Core Concepts EASY

What does the bias b do in the perceptron formula?

z = w₁·x₁ + w₂·x₂ + b
  • AIt scales the output between 0 and 1
  • BIt shifts the decision boundary without changing its orientation
  • CIt controls the learning rate
  • DIt normalizes the input features
ANSWER

B — Changing b translates the decision boundary parallel to itself. The weights w₁ and w₂ control the orientation (slope) of the boundary and are untouched by b.

Q 02 / 15 Core Concepts EASY

A perceptron uses w₁ = 1, w₂ = 1, b = −1.5. What is the output for input (x₁, x₂) = (1, 1)?

  • Az = −0.5 → output 0
  • Bz = +0.5 → output 1
  • Cz = 0 → output 0
  • Dz = +1.5 → output 1
ANSWER

B — z = 1×1 + 1×1 − 1.5 = +0.5. Since z ≥ 0, output = 1. This is the AND gate: both inputs are 1, so the perceptron fires.

Q 03 / 15 Core Concepts EASY

Which activation function outputs values in the range (0, 1), is smooth, and is differentiable everywhere?

  • AStep function
  • BSign function
  • CReLU
  • DSigmoid
ANSWER

D — The sigmoid σ(x) = 1/(1+e−x) maps all real numbers to (0,1) and is infinitely differentiable. This makes it essential for backpropagation. ReLU is not differentiable at 0; step and sign are discontinuous.

Q 04 / 15 Core Concepts MEDIUM

In vectorized notation, how do you compute predictions for N examples at once, where X is the N×2 input matrix?

  • Az = w * X + b (element-wise multiplication)
  • Bz = X @ w + b (matrix-vector product)
  • Cz = sum(w) + b
  • Dz = X * w.T + b
ANSWER

B — X @ w + b performs a matrix-vector product: each row of X (one example) is dotted with w, then b is broadcast-added. Result: a vector of N scores. This is the core of efficient batch computation in NumPy.

02

Logic Gates

NOT, AND, OR, NAND, XOR — Questions 5 to 8

Q 05 / 15 Logic Gates EASY

A perceptron with w₁ = 1, w₂ = 1, b = −0.5 receives input (0, 0). What is z and what is the output?

  • Az = −0.5, output = 0
  • Bz = +0.5, output = 1
  • Cz = 0, output = 1
  • Dz = −1.5, output = 0
ANSWER

A — z = 1×0 + 1×0 − 0.5 = −0.5. Since z < 0, output = 0. This is the OR gate: when both inputs are 0, output should be 0. ✓

Q 06 / 15 Logic Gates MEDIUM

You want to implement a NOT gate with a single-input perceptron. Which set of parameters is correct?

  • Aw₁ = 1, b = −0.5
  • Bw₁ = −1, b = 0.5
  • Cw₁ = 0, b = 1
  • Dw₁ = 1, b = 0.5
ANSWER

B — The negative weight w₁ = −1 inverts the input's effect. Check: NOT(0) = −1×0 + 0.5 = +0.5 ≥ 0 → 1 ✓ ; NOT(1) = −1×1 + 0.5 = −0.5 < 0 → 0 ✓.

Q 07 / 15 Logic Gates MEDIUM

The AND gate uses w₁ = 1, w₂ = 1, b = −1.5. How do you derive the NAND gate from it?

  • AAdd 1 to the bias
  • BSwap the two weights
  • CMultiply all parameters (w₁, w₂, b) by −1
  • DReplace the step function with the sign function
ANSWER

C — NAND = NOT AND. Multiplying all parameters by −1 gives w₁=−1, w₂=−1, b=+1.5. Now zNAND = −zAND, so every AND output is flipped. This is a general principle: NOT(f) uses parameters (−w, −b).

Q 08 / 15 Logic Gates MEDIUM

Which logic gate cannot be implemented by a single perceptron, regardless of the weight values chosen?

  • AOR
  • BAND
  • CXOR
  • DNAND
ANSWER

C — XOR is the classic example of a non-linearly separable function. Class 1 contains (0,0) and (1,1) on one diagonal; Class 0 contains (0,1) and (1,0) on the other. No straight line can separate them.

03

Geometry

Decision boundaries, slopes, linear separability — Questions 9 to 12

Q 09 / 15 Geometry MEDIUM

A perceptron has w₁ = 2, w₂ = −1, b = 0. Written as x₂ = f(x₁), what is the slope of the decision boundary?

  • A−2
  • B+2
  • C−1/2
  • D+1/2
ANSWER

B — From 2x₁ − x₂ = 0, isolate x₂: x₂ = −(w₁/w₂)·x₁ = −(2/−1)·x₁ = +2·x₁. Slope = +2. The negative w₂ flips the sign of the division, making the slope positive.

Q 10 / 15 Geometry MEDIUM

In a 2D perceptron, what happens to the decision boundary when you increase b?

  • AThe boundary rotates clockwise
  • BThe boundary shifts parallel to itself (translation)
  • CThe boundary becomes curved
  • DThe weights are rescaled proportionally
ANSWER

B — In x₂ = −(w₁/w₂)x₁ − b/w₂, changing b only affects the intercept −b/w₂. The slope −w₁/w₂ is determined solely by the weights. Increasing b shifts the line up/down (parallel translation).

Q 11 / 15 Geometry HARD

Points (0,0) and (1,1) are Class 1; (0,1) and (1,0) are Class 0 (XOR). Which argument proves no perceptron can solve this?

  • AThe points are too far apart for any line to reach both classes
  • BThe two classes are not convexly separable — their diagonals share the same midpoint (0.5, 0.5)
  • CThere are too many points for a 2-weight perceptron to handle
  • DThe bias would need to take an imaginary value
ANSWER

B — The centroid of Class 1 is (0.5, 0.5) and so is the centroid of Class 0. Any separating halfspace would need to put one side above (0.5,0.5) and the other below — but both centroids coincide. This is a formal proof of non-linear separability.

Q 12 / 15 Geometry MEDIUM

The decision boundary is 4x₁ + 2x₂ − 8 = 0. Which point lies exactly on the boundary?

  • A(1, 2)
  • B(2, 0)
  • C(0, 4)
  • D(1, 1)
ANSWER

B (and C) — Check (2,0): 4×2 + 2×0 − 8 = 8 − 8 = 0 ✓. Check (0,4): 4×0 + 2×4 − 8 = 8 − 8 = 0 ✓. Both B and C lie on the boundary. Check (1,2): 4+4−8=0 ✓ too! Points on the boundary have z = 0 → assigned to Class 1 by convention.

Analysis & Reasoning

Applied problems and theoretical understanding — Questions 13 to 15

Q 13 / 15 Analysis HARD

A student classifier uses w₁ = 2.0, w₂ = 0.5, b = −60. A student studied 8 hours and has a previous score of 70. Predict Pass or Fail.

  • Az = −9.0 → Fail
  • Bz = +9.0 → Pass
  • Cz = +26.0 → Pass
  • Dz = −26.0 → Fail
ANSWER — CAREFUL CALCULATION REQUIRED

A — z = 2.0×8 + 0.5×70 − 60 = 16 + 35 − 60 = −9 < 0 → Fail. Many students pick B by forgetting to apply the bias. Always compute the full z, including b!

Q 14 / 15 Analysis HARD

You test hundreds of weight combinations on a dataset. Accuracy never exceeds 94%. What is the most likely explanation?

  • AThe learning rate is too high
  • BThe activation function is wrong
  • CThe dataset is not linearly separable — the classes overlap
  • DThe bias is missing
ANSWER

C — When exhaustive search over all weight combinations cannot improve accuracy, the bottleneck is the model's capacity, not the parameters. The data is likely not linearly separable — the 6% error comes from overlapping classes that no single line can correctly separate.

Q 15 / 15 Analysis HARD

XOR can be solved by a 2-layer network. Which combination of perceptrons correctly implements XOR?

  • ANOT(AND(x₁, x₂))
  • BAND(OR(x₁, x₂), NAND(x₁, x₂))
  • COR(AND(x₁, x₂), NOT(x₁))
  • DNAND(OR(x₁, x₂), AND(x₁, x₂))
ANSWER

B — OR catches "at least one input is 1"; NAND catches "not both are 1"; AND of those two gives exactly "exactly one input is 1" = XOR. Verify: (0,0)→AND(0,1)=0✓ (1,0)→AND(1,1)=1✓ (0,1)→AND(1,1)=1✓ (1,1)→AND(1,0)=0✓.
