Perceptron Simple Resume
=======================
**Title:** Perceptron: Foundations and first implementations
**Content:**
- Biological motivation and mathematical formulation with LaTeX
- Geometric interpretation (decision boundaries, weight vectors)
- Detailed learning algorithm with convergence theorem
- Complete Python implementation from scratch

**Examples:**
- AND logic gate with visualizations
- Iris flower classification (real dataset)
- XOR problem demonstrating limitations

**Exercises (Progressive Difficulty):**

*Easy (3 exercises):*
- Manual calculations and geometric understanding
- OR and NAND gate implementations

*Medium (3 exercises):*
- Learning rate analysis
- Margin effects on convergence
- Pocket algorithm for non-separable data

*Hard (4 exercises):*
- Multi-class extension (one-vs-all)
- Convergence theorem verification
- Voted perceptron ensemble
- Kernel perceptron (challenge)

The notebook emphasizes mathematical rigor while maintaining pedagogical clarity, with all visualizations, working code examples, and space for students to complete exercises. Perfect for introducing neural networks through the lens of the perceptron family!
